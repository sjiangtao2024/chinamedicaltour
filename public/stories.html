Elite Health Package - Detailed Items

General Examination

  • Height, weight, blood pressure, BMI
  • Internal medicine, surgery, ophthalmology, ENT examination

Laboratory Tests

  • Complete blood count, urinalysis
  • Liver function (ALT, AST)
  • Kidney function (BUN, Cr)
  • Lipid profile (TC, TG, HDL, LDL)
  • Fasting glucose (GLU)

Instrumental Examination

  • Static ECG
  • Abdominal ultrasound (liver, gallbladder, pancreas, spleen, kidneys)
  • Chest X-ray

Platinum Package - Detailed Items

Includes all previous package items, plus:

Advanced Laboratory Tests

  • Complete liver function panel
  • Complete kidney function panel
  • Thyroid function (T3, T4, TSH)
  • Rheumatoid factors (RF, ASO, CRP)
  • Homocysteine (Hcy)

Tumor Marker Screening

  • CEA (broad spectrum)

  • Carotid ultrasound

Special Tests

  • Bone density test
